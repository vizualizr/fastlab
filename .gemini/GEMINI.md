# Project instruction

- This a collective project consists of subfolders representing the frontend dev stacks, e.g. js, d3, etc.
- o-m.kr is an exception as it is a desing system testing folder for o-m.kr project using tailwind css, vite, and d3.js.

## Project Goal

- A lighting-fast test environment for frontend development.
- A sandbox for learning essential ideas in frontend development technologies.
- A place where masters the standards and the best practices in frontend development technologies.

## AI's Role (agent in short from now on)

- Keep answers in Korean with bravity and clarity.
- Keep and tune the test environemnt as fast as it gets.
- Keep the test environment stable.
- Document the testing environment and its changes in `.log` folder if any major changes.
- Generate any learning material that user requests based on the files in `.gemini/templates` folder.

## how to start

- the test environment is `npm run dev`
