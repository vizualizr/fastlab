## 2. 핵심 대화 요약 (Conversation Summary)
- **D3 데이터 바인딩 원리**: `d3.axis`가 내부적으로 `ticks` 데이터를 추출하여 각 `.tick` 요소에 자동으로 바인딩(`.data()`)하기 때문에, 우리가 따로 데이터를 주지 않아도 `.attr(..., d => ...)`에서 `d`를 사용할 수 있음을 확인.
- **Time Scale의 선형성**: `d3.scaleTime()`은 밀리초 단위의 선형 스케일이므로, 달마다 일수가 달라 눈금 사이의 간격이 미세하게 차이 나는 것이 정상임을 이해.
- **D3 Scale의 이중적 성격**: `xScale` 객체가 함수(값을 픽셀로 변환)이면서 동시에 객체(설정 메서드 보유)인 자바스크립트의 함수 객체 특성을 파악.
- **D3 학습 전략**: D3의 높은 러닝커브와 낮은 가독성을 인정하고, 모든 것을 외우기보다 **"4~6단계 데이터 파이프라인"** 흐름을 머릿속에 넣고 템플릿을 수정하는 방식으로 접근하기로 함.
